leetcode每日一题 等式方程的可满足性 2020.6.8
发表于 2020-06-08| 更新于 2020-06-13|未分类
阅读量:

题目描述

给定一个由表示变量之间关系的字符串方程组成的数组，每个字符串方程 equations[i] 的长度为 4，并采用两种不同的形式之一：”a==b” 或 “a!=b”。在这里，a 和 b 是小写字母（不一定不同），表示单字母变量名。

只有当可以将整数分配给变量名，以便满足所有给定的方程时才返回 true，否则返回 false。

解答

解法一

这是一道并查集的题目，并查集主要用来解答需要并和查的问题，比如给定一些元素可以在一定条件下合并，最后检查两个元素是不是在一个集合中，这种就是并查集的问题。并查集的实现底层是数组，指向父节点。

主要有两个操作，

并：将a的父节点指向b的父节点

查：查找两个节点的父节点，while循环直到指向自己

这道题就是看做并查集，所有的相等的元素因为传递性可以被看做一个集合，然后最后查找不相等的是否违反了上面建立的集合，违反了就输出false。

方法二

在方法一的基础上增加路径压缩和按秩合并。

路径压缩，在查询的时候将路径进行压缩，有隔代压缩（把父亲节点指向父亲节点的父亲节点）和完全压缩（全部指向根节点，一般用递归）

按秩合并（让高度低的树指向高度低的树）

都用的话，复杂度接近o(1)，一般用一个就行。

方法三

还有一些非主流的方法。

将abcd的值置为1234….以此类推，出现一样的就把值设为一样，比如出现a=c，就把所有值为3的地方设为1，最后比较值是否一样。
