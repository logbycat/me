『C++OJ练习』水仙花数

Description 所谓“水仙花数”是指一个三位数，其各位数字立方和等于该数 Input 输入n，表示有n行 接下来每行输入k， k为三位数 Output 如果k为水仙花数，则输出YES,否则为NO Sample Input 1 153 Sample Output YES [cc lang=”java” tab_size=”2” lines=”40”]

#include
using namespace std;

int main()
{
    int n, k, k1, k2, k3;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> k;
        k3 = k % 10;
        k2 = ((k % 100) - k3)/10;
        k1 = (k - k3 - 10 * k2)/100;
        if (k1*k1*k1 + k2*k2*k2 + k3*k3*k3 == k)cout << "YES";
        else cout << "NO";
        cout << endl;
    }
    return 0;
}

[/cc] VB上做过无数次的题。。。问题在于K3,K2,K1的取模从高中pascal就开始用现在还老是错。。。可以归纳为一个循环“取模，相减，除以10”简单来说还是太菜了。。。
