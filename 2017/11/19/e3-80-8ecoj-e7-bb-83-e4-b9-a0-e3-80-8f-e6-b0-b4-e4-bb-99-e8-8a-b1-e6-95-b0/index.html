『C++OJ练习』水仙花数
发表于 2017-11-19| 更新于 2020-05-23|C++作业
阅读量:

Description 所谓“水仙花数”是指一个三位数，其各位数字立方和等于该数 Input 输入n，表示有n行 接下来每行输入k， k为三位数 Output 如果k为水仙花数，则输出YES,否则为NO Sample Input 1 153 Sample Output YES

​
​

VB上做过无数次的题。。。问题在于K3,K2,K1的取模从高中pascal就开始用现在还老是错。。。可以归纳为一个循环“取模，相减，除以10”简单来说还是太菜了。。。
